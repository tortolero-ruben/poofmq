# poofMQ Quickstart

Get started with the poofMQ API and official SDKs.

## API base URL

- **Local:** `http://localhost:8080` (or set `GO_API_HOST_PORT` in `.env`)
- **Configure in portal:** Use the Laravel portal settings for your environment’s API base URL.

## Node.js (TypeScript/JavaScript)

1. From repo root: `make sdk-generate`, then:

   ```bash
   cd sdks/node && npm install && npm run build
   ```

2. Use the client:

   ```javascript
   import { PoofmqClient } from "poofmq";

   const client = new PoofmqClient({
     baseUrl: process.env.GO_API_BASE_URL || "http://localhost:8080",
     apiKey: process.env.POOFMQ_API_KEY,
   });

   const res = await client.push("my-queue", "user.created", { user_id: "123" });
   console.log("Message ID:", res.messageId);

   const message = await client.pop("my-queue");
   if (message) console.log(message.envelope?.eventType, message.envelope?.payload);
   ```

See [sdks/node/README.md](../sdks/node/README.md) for options and client-side encryption.

## Python

1. From repo root: `make sdk-generate`, then:

   ```bash
   cd sdks/python && pip install ./generated && pip install -e .
   ```

2. Use the client:

   ```python
   from poofmq import PoofmqClient
   import os

   client = PoofmqClient(
       base_url=os.environ.get("GO_API_BASE_URL", "http://localhost:8080"),
       api_key=os.environ.get("POOFMQ_API_KEY"),
   )
   res = client.push("my-queue", "user.created", {"user_id": "123"})
   print("Message ID:", res.message_id)
   message = client.pop("my-queue")
   if message:
       print(message.envelope.event_type, message.envelope.payload)
   ```

See [sdks/python/README.md](../sdks/python/README.md) for options and client-side encryption.

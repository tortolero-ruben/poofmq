# ADRs

Architecture Decision Records are stored here.

## Naming convention

- `YYYY-MM-DD-<issue-or-topic>.md`
- Example: `2026-03-01-rub-240-redis-data-model-ttl-auth-namespace.md`

## Required sections

- Context
- Decision
- Security implications
- Consequences
- Implementation references (issue links/constants)

## Status values

- Proposed
- Accepted
- Superseded
- Deprecated
